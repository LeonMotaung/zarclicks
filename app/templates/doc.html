API Code Examples

Below are sample API requests to get you started. For complete endpoint details, visit x.ai/api.

Fetch Influencer Data


# Example: Retrieve influencers by category
curl -X GET "https://api.1nflou.com/v1/influencers" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"category": "fashion", "followers_min": 10000, "limit": 10}'

Create a Campaign


# Example: Create a new influencer campaign
curl -X POST "https://api.1nflou.com/v1/campaigns" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Summer Fashion Campaign",
    "start_date": "2025-10-01",
    "budget": 5000,
    "influencer_ids": ["123", "456"]
  }'

Retrieve Campaign Analytics


# Example: Get analytics for a campaign
curl -X GET "https://api.1nflou.com/v1/campaigns/789/analytics" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
